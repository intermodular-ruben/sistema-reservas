# Sistema de Reservas

## Proyecto Intermodular 1º DAM

Este repositorio contiene los entregables desarrollados durante el Proyecto Intermodular de 1º DAM, centrado en el diseño y desarrollo de un sistema de reservas de hotel.

El proyecto ha evolucionado desde el análisis y diseño inicial hasta la creación de una aplicación CRUD y una API REST básica como prueba de concepto (PoC).

---

## Tecnologías utilizadas

- Java
- HTML y CSS
- MySQL
- XAMPP
- API REST
- Git y GitHub

## Objetivo del proyecto

El objetivo del proyecto es desarrollar un sistema de reservas funcional que permita gestionar clientes, habitaciones y reservas, aplicando conocimientos adquiridos en programación, bases de datos, lenguajes de marcas y entornos de desarrollo.
